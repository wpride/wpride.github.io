Statter

An analysis tool for linear functions on arbitrary datasets

Today I released (and named) statter, a tool I originally designed to be make generation and analysis of baseball statistics accessible. The idea was that the user could produce his own equation over the set of baseball statistics and sort players by this statistic.

As I progressed with this tool I realized I was repeating a lot of work I'd done in previous data projects. By modularizing and generalization some parts of the code I was able to make the program applicable over an arbitrary dataset.

The next steps for this project are introducing some generics and an interface to handle non-Integer variables, adding a reasonable front end, and finally hosting the project as a jetty web app

Depending how this goes, I could envision a more expansive platform that allowed users to upload datasets, save equations over these datasets as reports, and share these reports with friends and publicly
